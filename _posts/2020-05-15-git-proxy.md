###git 使用代理命令设置
* git remote -v 查看当前远程库的信息
* git remote rm origin 删除远程仓库配置
* git remote add origin https://github.com/xxxx
* git config --global http.proxy http://xxx.com:port
* git config --global https.proxy http://xxx.com:port
